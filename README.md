*FocusTimer - Productivity & Task Management System
*FocusTimer - Verimlilik ve Görev Yönetim Sistemi
[English]
FocusTimer is a desktop application developed with C# and SQLite, designed to help users manage their daily tasks while following the Pomodoro technique.

Key Features:

Task Management (CRUD): Users can add, edit, delete, and search for tasks.

Data Visualization: Daily and weekly productivity levels are visualized using the Chart library.

User Authentication: Secure Login and Registration screens.

Customizable Settings: Adjustable study and break durations for the Pomodoro timer.

Technologies Used:

C# / Windows Forms (.NET)

SQLite Database

.NET Chart Controls

[Türkçe]
FocusTimer, kullanıcıların günlük görevlerini yönetmelerine ve Pomodoro tekniği ile verimliliklerini artırmalarına yardımcı olan, C# ve SQLite ile geliştirilmiş bir masaüstü uygulamasıdır.

Öne Çıkan Özellikler:

Görev Yönetimi (CRUD): Görev ekleme, düzenleme, silme ve arama fonksiyonları.

Veri Görselleştirme: Günlük ve haftalık verimlilik istatistikleri Chart kütüphanesi ile grafiklere dökülür.

Kullanıcı Yönetimi: Güvenli Kayıt Ol ve Giriş Yap ekranları.

Kişiselleştirilebilir Ayarlar: Kullanıcıya özel çalışma ve mola süreleri.

Kullanılan Teknolojiler:

C# / Windows Forms (.NET)

SQLite Veritabanı

.NET Chart Kontrolleri
